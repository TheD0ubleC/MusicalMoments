# MusicalMoments 使用教程（占位）

> 这是临时教程内容，后续可以直接替换为正式 Markdown 文档。

## 1. 快速开始

1. 下载并运行 `MusicalMoments`。
2. 在设置页完成音频输入/输出设备选择。
3. 绑定快捷键并选择默认播放音频。

## 2. 常见场景

### 直播场景

- 使用双通道输出，确保直播软件与本地监听都能收到音频。
- 推荐先使用稳定版验证配置，再尝试预览版新功能。

### 游戏场景

- 开启全局快捷键触发，提高战斗中的操作效率。
- 保持音频库结构清晰，避免误触。

## 3. 性能建议

- 根据设备性能调整缓冲区与预加载策略。
- 在高负载场景下优先选择稳定通道。
- 定期更新到新版本获得性能修复。

## 4. 插件与扩展

- 插件可通过下载页获取。
- 安装后建议重启应用以确保加载完整。

## 5. 故障排查

- 没有声音：检查输出设备与系统默认设备是否一致。
- 快捷键无效：尝试管理员权限运行应用。
- 音频卡顿：降低同时触发频率并调高缓冲区。
